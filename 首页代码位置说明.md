# 首页代码位置说明

文件路径：`src/views/videoIndex/index.vue`

---

## 轮播图

### 模板（Template）

| 位置 | 说明 |
|------|------|
| 约第 121 行 | `<!-- ========== 【轮播图区域】开始 ========== -->` 注释标记 |
| `<div class="hot-videos-carousel-wrapper">` | 轮播图外层容器，条件：`activeCategory === '首页' && !searchQuery` |
| `<el-carousel>` | Element UI 轮播组件，自动播放间隔 4000ms，高度 280px |
| `<el-carousel-item>` | 遍历 `hotVideoList`，点击调用 `goToVideoDetail(video.id)` |

### 数据 / 方法（Script）

| 位置 | 说明 |
|------|------|
| `data.hotVideoList` | 存储轮播图视频列表，初始为 `[]` |
| `data.hotVideoLoading` | 轮播图加载状态，初始为 `true` |
| `created()` → `this.fetchHotVideos()` | 页面创建时触发轮播图数据请求 |
| `fetchHotVideos()` 方法（约第 640 行） | 调用 `movieApi.getHotVideoInfo()` → `POST /admin/system/sysMovie/getHotVideoInfo`，将返回数据映射并经图片缓存后存入 `hotVideoList` |

### 样式（Style）

| 位置 | 说明 |
|------|------|
| `.hot-videos-carousel-wrapper` | 轮播图容器样式，含圆角、阴影 |
| `.carousel-item-content` | 单个轮播项，背景图铺满 |
| `.carousel-text-overlay` | 底部渐变遮罩，显示标题和播放量 |
| `.carousel-title` / `.carousel-meta` | 轮播文字样式 |
| `::v-deep .el-carousel__*` | 覆盖 Element UI 轮播指示器和箭头样式 |

## 排行榜

### 模板（Template）

| 位置 | 说明 |
|------|------|
| 约第 187 行 | `<!-- ========== 【排行榜侧边栏】开始 ========== -->` 注释标记 |
| `<aside class="sidebar">` | 排行榜侧边栏容器 |
| `<div class="rank-list">` | 遍历 `rankList`，点击调用 `goToVideoDetail(rank.id)` |
| `<span class="rank-num">` | 排名序号（1、2、3...） |
| `<span class="rank-title">` | 视频标题 |

### 数据 / 方法（Script）

| 位置 | 说明 |
|------|------|
| `data.rankList` | 存储排行榜列表，初始为 `[]` |
| `data.rankLoading` | 排行榜加载状态，初始为 `true` |
| `created()` → `this.fetchRankVideos()` | 页面创建时触发排行榜数据请求 |
| `fetchRankVideos()` 方法（约第 615 行） | 调用 `movieApi.getHotWatchVideoInfo()` → `POST /admin/system/sysMovie/getHotWatchVideoInfo`，返回按播放量排序的视频列表存入 `rankList` |

### 样式（Style）

| 位置 | 说明 |
|------|------|
| `.sidebar` | 侧边栏整体样式，固定宽度 |
| `.rank-list` | 排行榜列表容器 |
| `.rank-item` | 单条排行项，hover 高亮 |
| `.rank-num` | 序号样式，前三名有特殊颜色 |
| `.rank-title` | 标题文字，超出省略 |

## 接口对应关系

| 功能 | 接口方法 | 后端路径 |
|------|----------|----------|
| 轮播图数据 | `movieApi.getHotVideoInfo()` | `POST /admin/system/sysMovie/getHotVideoInfo` |
| 排行榜数据 | `movieApi.getHotWatchVideoInfo()` | `POST /admin/system/sysMovie/getHotWatchVideoInfo` |

接口定义文件：`src/api/movie/movie.js`
